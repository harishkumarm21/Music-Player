# 🎵 Web Music Player

A fully functional and stylish web-based music player built using **HTML**, **CSS**, and **vanilla JavaScript**.

![Web Music Player Screenshot](readme-images/desktop.png)

---

## 🚀 Features

- Elegant and responsive UI
- Play/Pause functionality
- Track progress bar
- Forward, backward, and repeat controls
- Metadata display (title, artist, year)
- Built purely with vanilla JavaScript — no frameworks

---

## 🧪 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📂 Project Structure

- `index.html` – Main entry point
- `assets/css/style.css` – Stylesheet
- `assets/js/script.js` – Application logic
- `assets/music/` – Local audio files
- `assets/images/` – Album artwork and icons
